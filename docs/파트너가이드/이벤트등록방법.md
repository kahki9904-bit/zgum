# 이벤트 등록 방법

## 등록 위치

파트너 페이지 (오른쪽으로 스와이프) → 하단 캡슐(이음) 위로 드래그

## 등록 순서

1. **제목 입력** — 이벤트 이름 (필수)
2. **사진 추가** — 최대 3장 (필수, 최소 1장)
   - 각 사진마다 설명 입력 가능
   - 대표 사진 선택 가능
3. **노출시간 선택** — 1시간 / 2시간 / 3시간
4. **등록 버튼** 탭
   - 19세 이상 이벤트 여부 확인 팝업
   - 위치 자동 수집 (현재 GPS 위치)
5. 등록 완료 → 지도에 마커 표시 + 패널에 카운트다운 시작

## 등록 후 관리

- 패널에서 남은 시간 확인
- **연장**: 1시간 단위 연장 가능 (무료이용 한도 내)
- **종료**: 패널 카운트다운 화면에서만 가능 (이벤트 상세 팝업에서는 불가)
- 시간 만료 → 자동 종료 → 파트너 페이지 피드에 기록 저장

## 무료이용 안내

- 첫 등록 후 **6개월간 무료** 이용
- 하루 등록 횟수 제한 있음 (한도 초과 시 등록 버튼 비활성화)
- 무료이용 현황은 파트너 대시보드에서 확인
