# 무료이용 안내

## 정책

- 파트너 **첫 이벤트 등록 시점부터 180일(6개월) 무료**
- 무료이용 기간 중 이벤트 등록·노출 비용 없음
- 하루 등록 횟수 제한 있음 (관리자 모드에서는 무제한)

## 무료이용 시작 조건

- 첫 이벤트 등록 완료 시 자동 시작
- 별도 신청 불필요

## 무료이용 종료 후

- 결제 연동 후 유료 전환 (Firebase 연동 후 구현)
- Naver Pay·Kakao Pay 지원 예정

## 알림 설정

- 무료이용 중 알림이 꺼져 있으면 이벤트 등록 시 알림 안내 팝업 표시
- 설정 → 알림에서 변경 가능
