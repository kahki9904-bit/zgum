# Z:GUM 사용자 기능 설명서

---

## Z:GUM이 뭔가요?

지금 이 순간, 내 주변 문화 행사를 발견하고, 직접 만들고, 기록하는 앱이에요.

보는 사람이 곧 만드는 사람이 될 수 있어요. 국가 공공 행사도, 동네 카페 팝업도, 버스킹 공연도 — 지금 당장 갈 수 있는 행사를 지도에서 확인하고, 내가 여는 행사는 바로 지도에 올릴 수 있어요. 회원가입 없이 시작할 수 있어요.

---

## 화면 구성

앱 아래에 탭 3개가 있어요.

- **흔적** — 내가 방문한 기록 모아보기
- **지금** — 주변 행사 지도로 보기
- **등록** — 내 행사를 지도에 올려 알리기

---

## 지금 — 주변 행사 찾기

앱을 켜면 내 위치 기준으로 주변 행사가 지도에 마커로 표시돼요.

직접 올린 현장 행사와 공공 문화 행사가 각각 다른 마커로 표시돼요. 마커를 탭하면 행사 이름, 장소, 시간, 내용을 볼 수 있어요.

**시간 표시** — 행사마다 남은 시간이 표시돼요. 마감이 가까울수록 마커가 빠르게 깜빡여요.

**필터** — 카테고리나 시간대로 걸러서 볼 수 있어요.

**검색** — 검색창에 행사 이름, 장소, 카테고리를 입력하면 원하는 행사 정보를 바로 찾을 수 있어요.

### 행사 알림

설정한 반경 안에 새 행사가 등록되면 알림이 와요. 알림을 받고 내용을 확인한 다음 갈지 말지는 내가 결정해요.

반경은 직접 설정할 수 있어요.
- **1km / 3km / 5km** 중 선택

---

## 등록 — 내 행사를 지금 지도에 올리기

카페 사장님도, 버스킹 뮤지션도, 핸드메이드 마켓 여는 작가도 — 지금 하는 행사를 올리면 근처 사람들 지도에 바로 표시돼요.

**등록 방법**
1. 행사 제목 입력
2. 사진 1~3장 촬영
3. 노출 시간 선택 (1시간 / 2시간 / 3시간)
4. 등록 완료 → 내 현재 위치에 지도에 즉시 표시

**진행 중 관리** — 남은 시간 카운트다운, 연장 또는 조기 종료 가능

---

## 흔적 — 방문 기록 남기기

행사에 방문했을 때 사진과 메모를 남기는 기능이에요.

행사 상세 화면에서 "여기 있어요" → 사진 촬영 → 메모 입력(선택) → 저장

방문한 행사들이 갤러리로 쌓여요.

---

## 이음 — 실제로 만난 사람과 연결

온라인에서 이름만 보고 팔로우 누르는 것과 달리, 이음은 **지금 이 자리에 같이 있는 사람**과만 연결할 수 있어요. 같은 공간에 있지 않으면 연결 자체가 불가능해요.

**연결 방법**
1. 내가 이음 요청
2. 100m 이내에 있는 상대방 폰에만 알림이 옴
3. 상대방이 기간을 선택하고 2자리 코드를 생성
4. 상대방이 코드를 **직접 말로** 알려줌
5. 내가 코드 입력 → 연결 완료

코드는 90초 안에 입력하지 않으면 사라져요. 그 자리에서 대화한 사람과만 연결되는 구조예요.

연결 기간은 1일 / 3개월 / 6개월 / 1년 중 선택할 수 있어요.

### 이음 알림 (핵심)

이음으로 연결된 사람이 설정한 반경 안에 들어오면 알림이 와요.

**"친구가 근처에 있어요. 찾아볼까요?"**

어디 있는지는 알려주지 않아요. 친구 목록도 따로 없어요. 감지됐다는 신호만 줄 뿐, 찾는 건 내가 직접 해야 해요.

반경은 직접 설정할 수 있어요.
- **100m / 300m / 500m** 중 선택

---

## 알림 설정

설정 화면에서 알림을 세밀하게 조정할 수 있어요.

| 알림 종류 | 내용 | 반경 설정 |
|---|---|---|
| 최신 이벤트 알림 | 설정 반경 안에 새 행사 등록 시 | 1km / 3km / 5km |
| 친구 감지 알림 | 이음 연결된 사람이 반경 안에 들어올 때 | 100m / 300m / 500m |
| 이음 완료 알림 | 이음 연결이 완료됐을 때 | — |
| 이음 만료 알림 | 연결 기간이 곧 끝날 때 | — |
