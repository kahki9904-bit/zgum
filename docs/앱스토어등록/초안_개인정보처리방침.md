## 1. 기본 원칙

Z:GUM은 별도의 회원가입을 요구하지 않습니다. 이름, 전화번호, 이메일, 주소 등 일반적인 회원가입 정보를 직접 입력받지 않는 것을 기본 방향으로 합니다.

다만 앱 기능을 제공하기 위해 위치 정보, 익명 사용자 식별자, 기기 식별용 임의 ID, 푸시 알림 토큰, 사용자가 직접 작성한 메모와 사진, 사용자가 직접 등록한 이벤트 정보, 문의 및 의견, 데이터 보존·복구를 위해 사용자가 선택적으로 등록한 이메일 주소 등이 처리될 수 있습니다.

Z:GUM은 회원가입 없이 사용할 수 있는 구조를 기본으로 합니다. 따라서 앱 삭제 또는 기기 변경 전 이메일을 미리 등록하지 않은 경우, 기존 기기와 연결된 데이터를 복구하지 못할 수 있습니다. 이메일 등록은 계정 가입이 아니라 데이터 보존과 복구를 위한 선택 기능입니다.

Z:GUM은 필요한 범위에서만 정보를 처리하며, 광고 추적이나 제3자 광고 목적으로 사용자 데이터를 판매하지 않습니다.

## 9. 아동 및 청소년 보호

Z:GUM은 성인 콘텐츠를 제공하지 않습니다. Z:GUM은 지역 문화 정보 탐색과 사용자 등록 이벤트 공유를 돕는 앱입니다.

다만 일부 현장 이벤트는 주최자 또는 등록자가 현장에서 참여 연령을 확인해야 하는 경우가 있을 수 있습니다. 이 경우 앱은 해당 이벤트에 연령 확인 필요 여부를 표시해 이용자가 미리 확인할 수 있도록 하는 1차 안전장치를 제공합니다.

현장에서 실제 입장 가능 여부, 연령 확인, 참여 제한 등은 해당 이벤트의 등록자 또는 현장 운영자가 직접 확인하고 안내해야 합니다.
