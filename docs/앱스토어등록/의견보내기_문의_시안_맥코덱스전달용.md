# Z:GUM 의견보내기 / 문의 시안 전달용

이 문서는 맥 코덱스가 의견보내기 및 문의 기능 작업 전에 참고할 시안 위치와 작업 의도를 빠르게 확인하기 위한 전달용 메모입니다.

## 참고 시안 파일

1. 설정 화면 문의 및 의견 시안
   - `docs/앱스토어등록/문의_의견_화면시안.png`
   - `docs/앱스토어등록/문의_의견_화면시안.html`

2. 이벤트 상세페이지 의견보내기 시안
   - `docs/앱스토어등록/이벤트상세_의견보내기_코드기준_시안.png`
   - `docs/앱스토어등록/이벤트상세_의견보내기_코드기준_시안.html`
   - `docs/앱스토어등록/이벤트상세_의견보내기_코드기준_시안.svg`

## 작업 의도

- 설정 화면에는 `문의 및 의견` 항목을 추가한다.
- 이벤트 상세페이지에는 기존 상세페이지 구조를 바꾸지 않고 `의견 보내기` 항목만 추가한다.
- `신고`라는 표현보다 `의견 보내기`, `문의 및 의견`처럼 부드러운 표현을 사용한다.
- 사용자가 작성한 내용은 운영자가 확인할 수 있는 데이터로 저장한다.
- 관리자 시각화 화면에서 문의/의견 확인 및 처리 상태를 볼 수 있게 연결한다.

## 주의

- 이벤트 상세페이지의 외형, 팝업 크기, 기존 흐름, `지금` 버튼 위치는 변경하지 않는다.
- 시안은 작업 방향 참고용이며, 실제 구현은 현재 앱 코드 구조에 맞춘다.
