## 1. 목적

본 약관은 Z:GUM(이하 "서비스")의 이용 조건, 서비스와 이용자의 권리·의무, 이용 제한, 책임 범위 등을 정하기 위한 것입니다.

Z:GUM은 사용자가 현재 위치 또는 선택한 위치를 기준으로 주변의 문화 행사, 공연, 공간, 사용자 등록 이벤트를 발견하고, 직접 이벤트를 등록해 주변 사용자에게 알리며, 방문 기록과 이음 기능을 이용할 수 있도록 돕는 위치 기반 모바일 서비스입니다.

## 2. 서비스의 주요 기능

서비스는 다음 기능을 제공합니다.

| 기능 | 설명 |
|---|---|
| 이벤트 등록 | 사용자가 직접 제목, 설명, 사진, 위치, 노출 시간을 입력해 나만의 이벤트를 등록하고 주변 사용자에게 알릴 수 있습니다. |
| 지도 기반 탐색 | 현재 위치 또는 선택 위치 기준으로 주변 문화 행사, 공연, 장소, 사용자 등록 이벤트를 표시합니다. |
| 검색 | 앱 내 행사 정보와 외부 장소 검색 결과를 함께 확인할 수 있습니다. |
| 흔적/체크인 | 사용자가 방문한 행사 또는 장소에 메모와 사진을 남길 수 있습니다. |
| 이음 | 가까운 거리의 사용자와 제한된 시간 동안 연결할 수 있는 기능입니다. |
| 등록 이벤트 관리 | 자신이 등록한 이벤트의 노출 상태, 사진, 시간, 통계 등을 확인할 수 있습니다. |
| 알림 | 사용자가 허용한 경우 이벤트, 이음, 등록 이벤트 등과 관련된 알림을 받을 수 있습니다. |
| 언어 설정 | 한국어, 영어, 일본어, 중국어 화면을 지원할 수 있습니다. |

## 3. 이용 조건

서비스는 별도의 회원가입 없이 이용할 수 있습니다. 다만 서비스 안정성, 알림, 기록 저장, 이벤트 등록, 이음 기능 제공을 위해 익명 식별자 또는 기기 기반 식별 정보가 생성될 수 있습니다.

일부 기능은 위치 권한, 알림 권한, 카메라/사진 접근 권한, 기기 인증 기능을 필요로 할 수 있습니다. 이용자는 기기 설정에서 권한을 허용하거나 거부할 수 있습니다. 권한을 거부하면 일부 기능이 제한될 수 있습니다.

사용자는 데이터 보존과 복구를 위해 이메일을 선택적으로 등록할 수 있습니다. 이메일 등록은 회원가입이 아니며, 앱 삭제 또는 기기 변경 전에 이메일을 등록하지 않은 경우 기존 데이터 복구가 제한될 수 있습니다.

## 4. 위치 기반 기능

서비스는 주변 행사와 장소를 보여주고, 사용자가 등록한 이벤트를 해당 위치 주변에 노출하기 위해 위치 정보를 사용합니다. 위치 권한이 허용되지 않거나 위치를 가져올 수 없는 경우, 이전에 저장된 위치 또는 기본 위치를 기준으로 서비스가 제공될 수 있습니다.

서비스의 위치 정보는 주변 탐색, 행사 표시, 이벤트 등록, 이음 요청 탐색 등 기능 제공을 위한 목적으로만 사용됩니다. Z:GUM은 위치 정보를 긴급 구조, 실시간 안전 보장, 교통·의료·재난 대응 용도로 제공하지 않습니다.

## 5. 이벤트 등록

이용자는 Z:GUM에서 직접 이벤트를 등록할 수 있습니다. 등록 이벤트는 주변 사용자에게 표시될 수 있으며, 등록자는 자신이 입력한 정보에 대한 책임을 집니다.

등록 가능한 정보에는 이벤트 제목, 설명, 사진, 위치, 노출 시간, 현장 연령 확인 필요 여부 등이 포함될 수 있습니다.

등록자는 다음 사항을 지켜야 합니다.

| 항목 | 내용 |
|---|---|
| 정확성 | 실제 존재하거나 진행 가능한 이벤트만 등록해야 합니다. |
| 권리 보유 | 사진, 문구, 상표, 장소 정보 등 등록 콘텐츠에 필요한 권리를 보유해야 합니다. |
| 위치 적정성 | 이벤트와 무관한 위치, 타인의 사유지, 위험한 장소를 허위로 등록해서는 안 됩니다. |
| 사진 등록 | 등록 이벤트의 현장성과 신뢰성을 위해 앨범에 저장된 기존 이미지는 사용할 수 없으며, 지금 현장에서 앱으로 직접 촬영한 사진을 기준으로 등록해야 합니다. |
| 연령 확인 안내 | Z:GUM은 성인 콘텐츠를 제공하지 않습니다. 다만 일부 현장 이벤트에서 주최자 또는 등록자가 참여 연령 확인을 해야 하는 경우, 앱에서 이를 미리 표시해 이용자가 확인할 수 있도록 해야 합니다. |
| 등록자 책임 | 등록한 이벤트의 내용, 사진, 위치, 시간, 안내 문구와 관련해 확인이나 수정, 문의 대응, 분쟁 대응이 필요한 경우 등록자가 성실히 책임지고 처리해야 합니다. |

운영자는 허위 이벤트, 과도한 광고성 이벤트, 부적절한 이벤트, 권리 침해 이벤트, 안전상 문제가 있는 이벤트를 확인한 경우 사전 통지 없이 숨김, 삭제, 이용 제한 등의 조치를 할 수 있습니다.

안정적인 운영과 무분별한 등록 방지를 위해 등록 횟수, 노출 시간, 연장 가능 횟수 등 이용 한도가 적용될 수 있습니다. 구체적인 이용 조건은 앱 내 안내와 운영 정책에 따라 제공됩니다.

## 6. 흔적 및 사용자 작성 콘텐츠

이용자는 행사 또는 장소에 대해 메모, 사진, 체크인 기록 등을 남길 수 있습니다. 이용자는 자신이 작성하거나 업로드한 콘텐츠에 대해 필요한 권리를 보유해야 하며, 타인의 권리나 법령을 침해하는 콘텐츠를 등록해서는 안 됩니다.

다음 콘텐츠는 등록할 수 없습니다.

| 금지 콘텐츠 | 예시 |
|---|---|
| 불법 또는 유해한 콘텐츠 | 범죄, 폭력, 불법 거래, 불법 촬영물 |
| 타인의 권리 침해 | 초상권, 저작권, 개인정보 침해 |
| 혐오·비방·괴롭힘 | 특정 개인 또는 집단에 대한 모욕, 차별, 위협 |
| 부적절한 선정성 콘텐츠 | 앱의 목적과 맞지 않거나 미성년자에게 부적절한 노골적인 내용 |
| 허위 정보 | 실제와 다른 행사, 장소, 혜택, 결제 정보 |

운영자는 위반 콘텐츠를 확인한 경우 사전 통지 없이 숨김, 삭제, 이용 제한 등의 조치를 할 수 있습니다.

## 7. 이음 기능

이음은 가까운 거리의 사용자 사이에서 제한된 시간 동안 연결을 만들 수 있는 기능입니다. 이음 요청에는 익명 사용자 ID, 요청 위치, 요청 시간, 선택한 연결 기간, 확인 코드 등이 사용될 수 있습니다.

이용자는 이음 기능을 타인의 동의 없는 추적, 괴롭힘, 감시, 위협, 만남 강요, 개인정보 수집 목적으로 사용해서는 안 됩니다. 이음 연결은 사용자의 자발적 선택에 따라 이루어져야 하며, 서비스는 오프라인 만남의 안전을 보장하지 않습니다.

## 8. 외부 정보와 제3자 서비스

서비스는 Google Maps, Firebase, 공공데이터 API, 공연 정보 API, 장소 검색 API 등 외부 서비스를 사용할 수 있습니다. 외부 서비스가 제공하는 지도, 장소, 공연, 행사 정보는 각 제공처의 사정에 따라 정확성, 최신성, 제공 범위가 달라질 수 있습니다.

Z:GUM은 외부 제공 정보의 오류, 누락, 지연, 중단에 대해 서비스가 합리적으로 통제할 수 없는 범위에서는 책임을 지지 않습니다.

## 9. 알림

서비스는 이용자가 알림을 허용한 경우 이벤트 종료 전 알림, 이음 관련 알림, 주변 등록 이벤트 알림, 서비스 안내 알림을 보낼 수 있습니다.

알림은 광고를 무작위로 보내기 위한 수단이 아니라, 사용자가 앱 경험을 놓치지 않도록 안내하기 위한 기능입니다. 이용자는 기기 설정 또는 앱 설정에서 알림을 변경할 수 있습니다.

## 10. 데이터 보존 및 복구

Z:GUM은 회원가입 없이 사용할 수 있는 앱입니다. 사용자의 기록은 익명 사용자 ID 또는 앱 내부 기기 ID와 연결될 수 있습니다.

사용자는 설정의 데이터 복구 화면에서 이메일을 미리 등록해 앱 삭제 또는 기기 변경 이후 데이터 복구에 사용할 수 있습니다. 이메일 등록 전 앱을 삭제하거나 기기를 변경한 경우 기존 데이터 복구가 제한될 수 있습니다.

## 11. 서비스 변경 및 중단

운영자는 서비스 개선, 보안, 정책 변경, 외부 API 변경, 서버 점검 등의 사유로 서비스의 일부 또는 전부를 변경하거나 중단할 수 있습니다. 중요한 변경이 있는 경우 앱 내 공지, 스토어 공지, 이메일 등 가능한 방법으로 안내합니다.

## 12. 이용 제한

운영자는 이용자가 다음 행위를 한 경우 서비스 이용을 제한할 수 있습니다.

| 제한 사유 | 내용 |
|---|---|
| 법령 위반 | 불법 정보 등록, 권리 침해, 사기성 행위 |
| 서비스 방해 | 비정상 접근, 자동화 도구, 서버 과부하 유발 |
| 오남용 | 이음 기능을 이용한 추적, 괴롭힘, 위협 |
| 허위 등록 | 존재하지 않는 이벤트, 허위 혜택, 허위 장소 등록 |
| 보안 위협 | 인증 우회, 데이터 조작, 악성 코드 사용 |

## 13. 개인정보 보호

서비스의 개인정보 처리에 관한 사항은 별도의 개인정보처리방침에 따릅니다. 이용자는 서비스 이용 전 개인정보처리방침을 확인해야 합니다.

## 14. 책임의 제한

운영자는 서비스가 제공하는 정보의 정확성과 안정성을 높이기 위해 노력합니다. 다만 외부 API 정보, 사용자 또는 등록자가 등록한 정보, 위치 오차, 네트워크 장애, 기기 설정 문제 등 운영자가 통제하기 어려운 사유로 발생한 손해에 대해서는 관련 법령이 허용하는 범위 내에서 책임이 제한됩니다.

이용자는 위치 기반 정보를 참고용으로 사용해야 하며, 실제 방문 전 이벤트 운영 여부, 장소, 시간, 비용, 입장 조건 등을 직접 확인하는 것이 좋습니다.

## 15. 약관의 변경

운영자는 필요한 경우 본 약관을 변경할 수 있습니다. 변경된 약관은 앱 내 공지 또는 별도 안내를 통해 고지하며, 변경 이후 서비스를 계속 이용하는 경우 변경된 약관에 동의한 것으로 볼 수 있습니다.
