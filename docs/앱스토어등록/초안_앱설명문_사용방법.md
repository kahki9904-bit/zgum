## 1. 한 줄 소개

Z:GUM은 지금 내 주변에서 열리는 문화 행사, 공연, 공간, 파트너 이벤트를 지도에서 발견하고 기록할 수 있는 위치 기반 문화 탐색 앱입니다.

## 2. 짧은 설명 후보

지금 내 주변의 문화 행사와 공간을 지도에서 발견하고, 나만의 흔적으로 남겨보세요.

## 3. 긴 설명 초안

Z:GUM은 "지금"을 기준으로 주변의 문화 경험을 발견하는 앱입니다.

앱을 열면 현재 위치 또는 선택한 위치를 기준으로 주변의 문화 행사, 공연, 장소, 파트너 이벤트가 지도에 표시됩니다. 사용자는 관심 있는 이벤트를 확인하고, 방문 후 흔적을 남기거나, 가까운 사람과 이음 기능으로 제한된 시간 동안 연결될 수 있습니다.

Z:GUM은 별도의 회원가입을 요구하지 않습니다. 위치 권한은 주변을 발견하기 위해 사용되며, 알림은 사용자가 놓치지 않도록 안내하는 목적으로만 사용됩니다.

## 4. 주요 기능

| 기능 | 설명 |
|---|---|
| 지금 주변 탐색 | 현재 위치 또는 선택한 위치 주변의 문화 행사와 장소를 지도에서 확인합니다. |
| 검색 | 행사명, 장소명 등을 검색해 원하는 정보를 빠르게 찾습니다. |
| 상세 정보 | 행사 제목, 장소, 시간, 사진, 설명, 파트너 메시지를 확인합니다. |
| 흔적 | 방문한 장소나 행사에 메모와 사진을 남깁니다. |
| 이음 | 가까운 거리에 있는 사용자와 2자리 확인 코드를 통해 제한된 시간 동안 연결합니다. |
| 파트너 이벤트 | 파트너가 직접 사진, 제목, 설명, 위치, 노출 시간을 등록할 수 있습니다. |
| 알림 | 허용한 경우 이벤트, 이음, 파트너 이벤트 관련 안내를 받을 수 있습니다. |
| 다국어 | 한국어, 영어, 일본어, 중국어 화면을 지원할 수 있습니다. |

## 5. 처음 사용하는 방법

1. 앱을 실행합니다.
2. 위치 정보 사용 안내를 확인합니다.
3. 위치 권한을 허용하면 현재 위치 기준으로 주변 정보가 표시됩니다.
4. 위치 권한을 허용하지 않아도 기본 위치 또는 수동 위치 기준으로 일부 기능을 이용할 수 있습니다.
5. 지도에서 마커를 눌러 행사와 장소 정보를 확인합니다.
6. 방문한 이벤트에는 흔적을 남길 수 있습니다.
7. 설정에서 알림, 언어, 앱 정보를 확인할 수 있습니다.

## 6. 지도 탐색 방법

지도 화면은 Z:GUM의 중심 화면입니다.

| 사용 방법 | 설명 |
|---|---|
| 현재 위치 보기 | 위치 권한이 허용되면 현재 위치 주변 정보를 보여줍니다. |
| 마커 선택 | 지도 위 마커를 누르면 상세 정보를 확인할 수 있습니다. |
| 검색 | 검색창에 2글자 이상 입력해 행사나 장소를 찾습니다. |
| 필터 | 카테고리나 시간 조건에 따라 보이는 정보를 조정할 수 있습니다. |
| 수동 위치 | 위치를 정확히 가져오지 못한 경우 지도에서 위치를 직접 설정할 수 있습니다. |

## 7. 흔적 남기기

흔적은 사용자가 방문한 행사나 장소를 나중에 다시 볼 수 있도록 남기는 개인 기록입니다.

| 항목 | 설명 |
|---|---|
| 메모 | 방문 느낌이나 짧은 기록을 남길 수 있습니다. |
| 사진 | 카메라로 촬영한 사진을 함께 저장할 수 있습니다. |
| 기록 보기 | 사용자 방에서 최근 흔적과 이전 흔적을 확인할 수 있습니다. |
| 삭제 | 필요 없는 흔적은 삭제할 수 있습니다. |

## 8. 이음 사용 방법

이음은 가까운 거리에 있는 사용자끼리 제한된 기간 동안 연결하는 기능입니다.

1. 사용자 방에서 이음 버튼을 누릅니다.
2. 가까운 거리에 다른 사용자의 이음 요청이 있으면 수락 화면이 표시됩니다.
3. 요청이 없으면 새로운 이음 요청을 만들 수 있습니다.
4. 상대방이 연결 기간을 선택하면 2자리 확인 코드가 생성됩니다.
5. 요청자가 코드를 입력하면 이음 연결이 완료됩니다.
6. 연결은 사용자가 선택한 기간 동안만 유지됩니다.

이음은 상대방의 동의 없이 추적하거나 감시하기 위한 기능이 아닙니다. 안전한 사용을 위해 실제 만남이나 개인정보 교환은 신중하게 결정해야 합니다.

## 9. 파트너 이벤트 등록 방법

파트너는 앱 안에서 직접 이벤트를 등록할 수 있습니다.

1. 파트너 영역으로 이동합니다.
2. 이벤트 제목과 설명을 입력합니다.
3. 사진을 촬영하거나 등록합니다.
4. 위치를 확인합니다.
5. 노출 시간을 선택합니다.
6. 등록하면 지도와 파트너 화면에 이벤트가 표시됩니다.

현재 파트너 이벤트 등록은 무료이용 형태로 제공될 수 있습니다. 향후 유료 기능 또는 결제 기능이 추가될 경우 별도로 안내합니다.

## 10. 알림 사용 방법

알림을 허용하면 Z:GUM의 주요 안내를 받을 수 있습니다.

| 알림 종류 | 설명 |
|---|---|
| 이벤트 알림 | 사용자가 지정한 이벤트와 관련된 알림 |
| 이음 알림 | 이음 요청, 연결, 근접 안내 |
| 파트너 이벤트 알림 | 파트너 이벤트 관련 안내 |
| 서비스 알림 | 앱 이용에 필요한 안내 |

알림은 기기 설정 또는 앱 설정에서 언제든 변경할 수 있습니다.

## 12. 스토어 심사 메모용 요약

Z:GUM은 위치 기반 문화 탐색 앱입니다. 별도 회원가입 없이 Firebase 익명 인증을 사용하며, 사용자의 위치는 주변 행사 표시, 파트너 이벤트 등록, 이음 요청 탐색에 사용됩니다. 사용자가 직접 남긴 흔적 메모와 사진, 파트너 이벤트 사진은 Firebase Firestore/Storage에 저장될 수 있습니다.

지도는 Google Maps를 사용하며, 행사 정보는 공공데이터 API, KOPIS, 파트너 등록 데이터 등을 통해 표시됩니다. 알림은 Firebase Cloud Messaging과 로컬 알림 기능을 사용합니다.
