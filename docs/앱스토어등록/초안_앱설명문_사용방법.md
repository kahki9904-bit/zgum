## 1. 한 줄 소개

Z:GUM은 누구나 지금 주변에 알리고 싶은 이벤트를 직접 등록하고, 주변 사용자는 그 이벤트와 문화 행사, 공연, 공간을 지도에서 발견할 수 있는 위치 기반 문화 이벤트 앱입니다.

## 2. 짧은 설명 후보

내 주변 이벤트를 발견하고, 나만의 이벤트를 직접 등록해 지금 주변 사용자에게 알려보세요.

## 3. 긴 설명 초안

Z:GUM은 "지금"을 기준으로 주변의 문화 경험과 사용자 등록 이벤트를 연결하는 앱입니다.

앱을 열면 현재 위치 또는 선택한 위치를 기준으로 주변의 문화 행사, 공연, 장소, 사용자 등록 이벤트가 지도에 표시됩니다. 사용자는 관심 있는 이벤트를 확인할 수 있고, 직접 이벤트를 등록해 주변 사용자에게 알릴 수도 있습니다.

등록자는 사진, 제목, 설명, 위치, 노출 시간을 입력해 나만의 이벤트를 만들 수 있습니다. 주변 사용자는 지도와 알림을 통해 새로 등록된 이벤트를 발견하고, 방문 후 흔적을 남기거나, 가까운 사람과 이음 기능으로 제한된 시간 동안 연결될 수 있습니다.

Z:GUM은 별도의 회원가입을 요구하지 않습니다. 위치 권한은 주변을 발견하고 이벤트를 등록하기 위해 사용되며, 알림은 사용자가 놓치지 않도록 안내하는 목적으로만 사용됩니다.

회원가입이 없는 앱 특성상 앱 삭제 또는 기기 변경 전에 이메일을 미리 등록하지 않으면 기존 데이터를 복구하지 못할 수 있습니다. 이메일 등록은 계정 가입이 아니라, 사용자가 자신의 기록을 계속 이어가기 위해 선택하는 데이터 보존 기능입니다.

Z:GUM은 개발 경험이 없던 40대 후반의 개인이 AI 도구와 바이브 코딩 방식으로 직접 만들어낸 위치 기반 이벤트 앱입니다. 화면 안에 머무르는 정보 소비를 넘어, 사용자가 지금 있는 장소에서 실제로 움직이고 참여하도록 돕는 것을 목표로 합니다. 또한 작은 가게, 개인 행사, 현장 이벤트처럼 기존 광고와 팔로워 중심 홍보가 닿기 어려운 곳에 새로운 소규모 홍보 경로를 제안합니다.

## 5. 처음 사용하는 방법

1. 앱을 실행합니다.
2. 위치 정보 사용 안내를 확인합니다.
3. 위치 권한을 허용하면 현재 위치 기준으로 주변 정보가 표시됩니다.
4. 위치 권한을 허용하지 않아도 기본 위치 또는 수동 위치 기준으로 일부 기능을 이용할 수 있습니다.
5. 지도에서 마커를 눌러 행사, 장소, 등록 이벤트 정보를 확인합니다.
6. 직접 알리고 싶은 이벤트가 있다면 이벤트 등록 영역에서 사진, 제목, 설명, 위치, 노출 시간을 입력합니다.
7. 방문한 이벤트에는 흔적을 남길 수 있습니다.
8. 기록을 계속 보관하고 싶다면 앱을 삭제하거나 기기를 바꾸기 전에 설정의 데이터 복구 화면에서 이메일을 미리 등록합니다.
9. 설정에서 알림, 언어, 문의 및 의견, 앱 정보를 확인할 수 있습니다.

## 6. 이벤트 등록 방법

Z:GUM의 핵심 기능은 사용자가 직접 이벤트를 등록해 주변 사용자에게 알리는 것입니다.

1. 이벤트 등록 영역으로 이동합니다.
2. 이벤트 제목을 입력합니다.
3. 필요한 경우 설명 또는 메시지를 입력합니다.
4. 앱에서 현장 사진을 직접 촬영합니다.
5. 현재 위치 또는 선택한 위치를 확인합니다.
6. 이벤트가 노출될 시간을 선택합니다.
7. 현장에서 연령 확인이 필요한 이벤트인지 확인합니다.
8. 등록하면 해당 위치 주변 지도와 이벤트 화면에 표시될 수 있습니다.

등록 이벤트는 실제 주변 사용자에게 보여질 수 있으므로, 허위 정보, 권리 침해 사진, 부적절한 내용, 위험한 장소 정보는 등록하지 않아야 합니다. 허위성 또는 단순 광고성 등록을 줄이기 위해 등록 이벤트에는 앨범에 저장된 기존 이미지를 사용할 수 없으며, 지금 현장에서 앱으로 직접 촬영한 사진을 기준으로 합니다.

Z:GUM은 성인 콘텐츠를 제공하지 않습니다. 다만 일부 현장 이벤트에서 연령 확인이 필요한 경우, 등록자는 앱에 이를 표시하고 현장에서 직접 확인해야 합니다. 이 표시는 이용자가 미리 확인할 수 있도록 돕는 1차 안전장치입니다.

안정적인 운영과 무분별한 등록 방지를 위해 등록 횟수, 노출 시간, 연장 가능 횟수 등 이용 한도가 적용될 수 있습니다. 구체적인 이용 조건은 앱 내 안내와 운영 정책에 따라 제공됩니다.

## 8. 흔적 남기기

흔적은 사용자가 방문한 행사나 장소를 나중에 다시 볼 수 있도록 남기는 개인 기록입니다.

| 항목 | 설명 |

| 사진 | 카메라로 촬영한 사진을 함께 저장할 수 있습니다. |
| 기록 보기 | 사용자 방에서 최근 흔적과 이전 흔적을 확인할 수 있습니다. |
| 삭제 | 필요 없는 흔적은 삭제할 수 있습니다. |

## 9. 이음 사용 방법

이음은 가까운 거리에 있는 사용자끼리 제한된 기간 동안 연결하는 기능입니다.

1. 사용자 방에서 이음 버튼을 누릅니다.
2. 가까운 거리에 다른 사용자의 이음 요청이 있으면 수락 화면이 표시됩니다.
3. 요청이 없으면 새로운 이음 요청을 만들 수 있습니다.
4. 상대방이 연결 기간을 선택하면 2자리 확인 코드가 생성됩니다.
5. 요청자가 코드를 입력하면 이음 연결이 완료됩니다.
6. 연결은 사용자가 선택한 기간 동안만 유지됩니다.

이음은 상대방의 동의 없이 추적하거나 감시하기 위한 기능이 아닙니다. 안전한 사용을 위해 실제 만남이나 개인정보 교환은 신중하게 결정해야 합니다.

## 11. 데이터 보존 및 복구

Z:GUM은 회원가입 없이 사용할 수 있는 앱입니다. 그래서 사용자가 앱을 삭제하거나 기기를 바꾸기 전에 이메일을 미리 등록해두면 기존 데이터를 복구하는 데 사용할 수 있습니다.

| 항목 | 설명 |
|---|---|
| 데이터 보존 | 설정 → 데이터 복구에서 이메일을 등록합니다. 이메일 링크를 탭하면 등록이 완료됩니다. |
| 데이터 복구 | 새 기기 또는 재설치 후 이전에 등록한 이메일로 복구 링크를 받아 데이터를 연결합니다. |
| 주의 사항 | 이메일을 미리 등록하지 않은 상태에서 앱을 삭제하거나 기기를 변경하면 기존 데이터 복구가 어려울 수 있습니다. |

## 14. 스토어 심사 메모용 요약

Z:GUM은 위치 기반 문화 이벤트 앱입니다. 별도 회원가입 없이 Firebase 익명 인증을 사용하며, 사용자의 위치는 주변 행사 표시, 사용자 이벤트 등록, 이음 요청 탐색에 사용됩니다. 사용자가 직접 등록한 이벤트의 제목, 설명, 지금 현장에서 앱으로 직접 촬영한 사진, 위치, 노출 시간은 주변 사용자에게 표시될 수 있습니다.

사용자가 직접 남긴 흔적 메모와 사진, 등록 이벤트 사진은 Firebase Firestore/Storage에 저장될 수 있습니다. 데이터 보존과 복구를 위해 사용자가 선택적으로 이메일을 등록할 수 있으며, 문의 및 의견 화면에서 보낸 내용은 문의 처리와 오류 확인을 위해 저장될 수 있습니다. 지도는 Google Maps를 사용하며, 행사 정보는 공공데이터 API, KOPIS, 사용자 등록 데이터 등을 통해 표시됩니다. 알림은 Firebase Cloud Messaging과 로컬 알림 기능을 사용합니다.
