# Z:GUM 등록 기능 설명서

---

## 등록 기능 — 내 행사 올리고 관리하기

### 행사 등록
1. 행사 제목 입력
2. 사진 1~3장 촬영
3. 노출 시간 선택 (1시간 / 2시간 / 3시간)
4. 등록 완료 → 내 현재 위치에 지도에 즉시 표시

### 진행 중 관리
- 남은 시간 카운트다운 확인
- 시간 연장 또는 조기 종료 가능

### 현황 대시보드

**무료이용**
- 무료이용 활성 여부
- 오늘 등록 가능 여부
- 무료이용 잔여 일수

**기본 현황**
- 진행 중인 행사 수
- 종료된 행사 수
- 누적 방문 수
- 누적 흔적 수

**확장 데이터 (준비 중)**
- 상세 조회 데이터
- 지역 / 시간대 데이터
- by Z:GUM 확장 데이터
- 문화 공백 지역 분석
- 외국인 사용자 현황
