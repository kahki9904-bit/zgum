# Z:GUM 알림 시스템 정리

## 알림 항목

| 항목 | 푸시 알림 | 앱 내 알림 |
|------|----------|----------|
| 주변 새 이벤트 등록 | O (미연결) | - |
| 친구 근접 감지 | O (미연결) | - |
| 내 이벤트 종료 임박 (파트너용) | O (미연결) | - |

## 알림 설정 UX

- 알림 허용 토글 하나만 유지 (항목별 토글 불필요 — 기기에서 개별 조절 가능)
- **켜기** → 기기 설정으로 이동 (권한은 기기에서만 허용 가능)
- **끄기** → 앱 내 자체 비활성화 처리 + 팝업으로 안내

## 끄기 팝업 안내 내용

알림을 끄면 아래 기능을 받을 수 없습니다.

- 주변 새 이벤트 등록 알림
- 친구가 근처에 있어도 알 수 없음
- 내 이벤트 종료 임박 알림 (파트너)

## 삭제 대상

- 이벤트 종료 임박 알림 (일반 사용자용)
- 알림 캡슐 코드 (추후 확인 후 유지 여부 결정)
- 친구 등록·만료 알림
