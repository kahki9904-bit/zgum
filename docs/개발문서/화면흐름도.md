# 화면 흐름도

## 앱 진입 흐름

```
앱 실행
  ↓
브랜딩 화면 (Z:GUM 로고)
  ↓
[ConsentGuard - 비활성화 중, 출시 전 재연결]
  본인 인증 안내 → 나중에 / 인증하기
  위치 동의 → 미동의 / 동의
  ↓
Shell (메인 화면)
```

## 메인 화면 구조 (Shell)

```
← 스와이프 →
[사용자 페이지]  [지도 (기본)]  [파트너 페이지]

공통: 하단 캡슐 (이음) — 드래그하면 지금 패널 오픈
```

## 지금 패널 (캡슐 드래그)

```
캡슐 위로 드래그
  ↓
탭별 패널 내용 표시
  ├── 사용자 탭: 흔적 카드 + 이음 버튼
  ├── 지도 탭: 이벤트 목록 (피처드 + 나머지)
  └── 파트너 탭:
        ├── 진행 중 이벤트 없음 → 등록 폼
        └── 진행 중 이벤트 있음 → 카운트다운 + 연장 버튼
```

## 사용자 페이지

```
사용자 페이지 (왼쪽)
  ├── 흔적 피드 (1열 인스타 방식)
  │     탭 → 흔적 상세 팝업
  │     꾸욱 누르기 → 사진 팝업
  └── 우상단 설정 버튼 → 설정 화면
        ├── 언어·알림·성인인증·앱정보
        └── (관리자 모드 제거됨)
```

## 파트너 페이지

```
파트너 페이지 (오른쪽)
  ├── 이벤트 피드 (1열)
  │     탭 → 이벤트 상세 팝업 (사진 스와이프·통계·시간)
  └── 우상단 현황 버튼 → 파트너 대시보드
        ├── 무료이용 현황
        ├── 방문·흔적 통계
        └── 하단 관리자 모드 토글 (개발자 섹션)
```

## 체크인·흔적 흐름

```
지도에서 이벤트 핀 탭
  ↓
이벤트 상세 시트 (슬라이드 업)
  ↓
체크인 버튼
  ↓
카메라/갤러리 선택 → 사진 촬영
  ↓
메모 입력 (선택, 최대 100자)
  ↓
흔적 저장 → 사용자 페이지 피드에 추가
```

## 이음 (친구 연결) 흐름

```
패널 → 이음 버튼
  ↓
이음 소개 팝업 (최초 1회)
  ↓
이음 신청 팝업 (내 위치 전송)
  ↓
상대방 앱에서 수락 팝업
  ↓
친구 등록 완료
```
