# Z:GUM 관리자 기능 설명서

---

## 관리자 등급

| 등급 | 역할 |
|---|---|
| **총괄** | 앱 전체 권한, 이스터에그로 진입 |
| **관리자** | 총괄이 지정, 웹에서만 작업 |

---

## 계정 관리

사용자와 등록자 계정 상태를 조회하고 변경할 수 있어요.

**계정 상태 종류**
- 활성 — 정상 이용 중
- 정지 — 관리자 조치로 이용 제한
- 탈퇴 처리중 — 데이터 보존 기간 경과 후 삭제

**가능한 작업**
- 사용자·등록자 상태 조회
- 상태 변경 (활성 / 정지 / 탈퇴 처리)
- 정지된 계정 목록 조회

---

## 감사 로그

관리자가 어떤 작업을 했는지 전부 기록돼요.

**기록 대상**
- 이벤트
- 사용자 계정
- 등록자 계정
- 글로벌 설정
- 필터 정책
- 권한

**조회 방법**
- 최근 로그 전체 조회
- 대상별 조회
- 관리자별 조회

---

## 필터 정책

앱에 표시되는 콘텐츠 필터 기준을 설정하고 실시간으로 반영할 수 있어요.

---

## 글로벌 설정

앱 전체에 적용되는 설정을 조회하고 변경할 수 있어요. 변경 사항은 실시간으로 반영돼요.

---

## 관리자 전용 분석 (준비 중)

대시보드에서 관리자 전용 데이터 분석 항목이 별도로 제공될 예정이에요.
