# 파트너 영업 자료

## Z:GUM이 파트너에게 제공하는 것

- 앱 설치 사용자에게 현재 위치 기반으로 내 행사가 노출됨
- 별도 앱 개발·SNS 팔로워 없이도 새로운 방문객 유입 가능
- 방문·흔적 통계 제공 (얼마나 많은 사람이 왔는지 확인 가능)
- **첫 6개월 무료** — 비용 부담 없이 시작 가능

## 파트너 적합 대상

- 카페·갤러리·공방·식당 등 공간 보유 소상공인
- 팝업 전시·공연·버스킹·나눔장터 운영 개인 예술가
- 지역 행사를 알릴 채널이 없는 모든 사람

## 등록 방법 (3단계)

1. 앱 설치 → 파트너 페이지 이동
2. 하단 캡슐 위로 드래그 → 제목·사진·시간 입력
3. 등록 완료 → 지도에 즉시 표시

## Z:GUM ZONE (2단계)

지역 축제 개최 구역 전체를 Z:GUM ZONE으로 설정.
축제장 입장 방문객의 스마트폰에 참가 소상공인·예술가 부스 지도가 자동 표시.

| 대상 | 제공 가치 |
|---|---|
| 축제 주최 측 | 참가 업체 디지털 안내도 자동 생성, 방문객 행동 데이터 |
| 참가 소상공인 | 별도 비용 없이 방문객 스마트폰에 노출 |
| 방문객 | 축제 전체 지도 한눈에 파악, 방문 기록 자동 저장 |
