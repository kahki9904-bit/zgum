# 서비스 소개서

## Z:GUM (지금)

> 지역 문화 행사를 실시간으로 발견하고, 소상공인과 개인 예술가가 직접 알릴 수 있는 위치 기반 모바일 플랫폼

---

## 시장 현황

- 국내 문화 행사 시장: 연간 약 4조 원 규모 (문체부 2024)
- 공공 문화시설 연간 이용객: 1억 4천만 명 이상
- 국내 소상공인 수: 약 580만 명 — 대부분 디지털 마케팅 수단 부재
- **동일 앱 없음**: 위치 기반 실시간 문화 행사를 소상공인 등록·공공데이터가 동시에 표시되는 앱은 국내외 존재하지 않음

---

## 핵심 차별점 3가지

1. **공공 + 민간 통합**: TourAPI·KOPIS 공공 행사와 소상공인 직접 등록 행사가 하나의 지도에 표시
2. **등록 즉시 노출**: 소상공인이 앱에서 사진·제목·시간 설정 후 등록하면 지도에 바로 표시
3. **흔적 아카이빙**: 방문 인증·사진·메모가 사용자 개인 문화 생활 기록으로 저장

---

## 비즈니스 모델

| 수익원 | 방식 | 시점 |
|---|---|---|
| 행사 등록 구독 | 소상공인·개인 월정액 등록·노출 | 런칭 직후 |
| 프리미엄 노출 | 지도 상단 고정, 우선 노출 | 성장 후 |
| 지자체 B2B | 지역 문화재단·축제 운영기관 파트너십 | 2단계 |
| Z:GUM ZONE | 축제 구역 디지털 안내도 서비스 계약 | 2단계 |
| 데이터 인사이트 | 지역 문화 소비 패턴 리포트 | 장기 |

## 개발 현황

Flutter 기반 Android/iOS 크로스플랫폼 앱 MVP 완성

- 프론트엔드: 90% 완성
- 공공 API (TourAPI·KOPIS·SDSC·Kakao Local): 연동 완료
- Firebase·결제: 미연결 (구조 설계 완료)
- 출시 기준 달성도: 약 50~55%
