# 이음 (친구 연결)

## 이음이란

근거리(100m 이내)에 있는 사람과 연결하는 기능. 같은 행사에 온 사람과 자연스럽게 이어질 수 있다.

## 이음 신청 방법

1. 하단 캡슐 위로 드래그 → 지금 패널 오픈
2. '이음' 버튼 탭
3. 이음 소개 팝업 확인 (최초 1회)
4. 이음 신청 → 내 위치 전송

## 이음 수락

- 상대방 앱에 수락 팝업 자동 표시
- 수락하면 친구 목록에 추가

## 현황 확인

- 지금 패널에서 "N명과 이어졌습니다" 표시

## 주의

- 100m 이내에 있어야 신청 가능
- Firebase 연동 후 실시간 동기화 구현 예정 (현재 로컬 저장)
