# 시작하기

## 앱 설치 후 첫 실행

1. **브랜딩 화면** — Z:GUM 로고 표시 후 자동 진행
2. **위치 동의** — "주변을 발견하는 역할만 합니다."
   - 동의: 현재 위치 기반 행사 탐색 가능
   - 미동의: 기본 위치(서울시청)로 실행
3. **지도 화면** — 즉시 주변 행사 확인 가능

## 회원가입 불필요

- 설치 즉시 사용 가능
- 개인정보 수집 없음

## 화면 구성

```
← 스와이프           스와이프 →
[내 기록]      [지도]      [파트너]

하단 캡슐(이음) — 위로 드래그하면 지금 패널 오픈
```

- **지도** (기본 화면): 주변 행사 탐색
- **내 기록** (왼쪽): 내가 남긴 흔적 피드
- **파트너** (오른쪽): 행사 등록·관리
