# 지도·이벤트 탐색

## 지도 화면

앱 기본 화면. 현재 위치 기반으로 주변 행사가 마커로 표시된다.

## 행사 마커 종류

- **공공 행사**: 한국관광공사 TourAPI·KOPIS에서 자동 수집
- **민간 등록**: 소상공인·개인이 직접 등록한 행사
- 두 종류가 같은 지도에 함께 표시됨

## 행사 시간 표시 기준

- 30분~3시간 후 시작 예정 → 추천 표시
- 0~60분 남음 → 긴박 펄스 효과
- 30분 미만 남음 → 표시 제외

## 탐색 방법

1. 지도에서 핀 탭 → 행사 상세 시트 오픈
2. 장소·시간·내용 확인
3. 체크인 또는 알림 설정

## 검색

- 상단 검색창 → 카카오 로컬 API 장소 검색
- 검색 시 `--dart-define=KAKAO_API_KEY` 포함 빌드 필요

## 필터

- 카테고리별 필터 (공연·전시·축제 등)
- 시간 필터 (오늘·내일·이번 주)
