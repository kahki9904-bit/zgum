# 체크인·흔적 남기기

## 체크인이란

행사를 방문한 기록. 사진과 메모를 남겨 내 문화 생활을 아카이빙할 수 있다.

## 체크인 방법

1. 지도에서 행사 핀 탭 → 상세 시트 오픈
2. 체크인 버튼 탭
3. 카메라 또는 갤러리에서 사진 선택 (선택)
4. 메모 입력 (선택, 최대 100자)
5. 저장 완료

## 흔적 확인

- 내 기록 페이지 (왼쪽 스와이프) → 1열 피드 형식
- 최신순 / 과거순 정렬 변경 가능
- 카드 탭 → 상세 팝업 (날짜·시간·장소·메모 전체 확인)
- 카드 꾸욱 누르기 → 사진 팝업 (사진만 크게 보기)

## 흔적 삭제

- 지금 패널 (캡슐 드래그) → 흔적 카드 아래 '잊기' 버튼
