# 핵심 가치·철학

## 설계 원칙

- **모바일 전용**: 태블릿·PC 대응 없음. 스마트폰 사용자를 위한 최적화에 집중
- **화이트 + 딥블루**: 단순하고 깔끔한 색상 체계. 불필요한 장식 없음
- **단순함**: 기능보다 흐름. 사용자가 생각 없이 움직일 수 있는 UX

## 사회적 가치

- **지역 경제 활성화**: 소상공인·예술가의 홍보 비용 절감, 신규 방문객 연결
- **문화 소비 균형**: 대형 플랫폼 중심에서 지역·소규모 공간으로 분산
- **청소년 문화 접근성**: 분산된 정보를 하나의 앱에서 실시간 제공
- **청소년 안전 보호**: 성인 인증 기반 콘텐츠 분리 — 미성년자에게 성인 콘텐츠 구조적 차단
- **개인정보 보호 우선**: 회원가입 없이 설치 즉시 사용, 식별 정보 외 개인정보 수집 없음
- **공공데이터 활용**: 공개된 공공 행사 데이터가 시민에게 실제로 닿는 구조 제시
