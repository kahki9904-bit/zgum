# 주요 기능 목록

## 완료된 기능

| 기능 | 설명 |
|---|---|
| 위치 기반 행사 지도 | GPS 현재 위치 기준 반경 내 행사 실시간 마커 표시 |
| GPS 4단계 폴백 | GPS 정밀 → 이전 세션 위치 → Wi-Fi/기지국 → 기본값(서울시청) |
| 공공 API 연동 | TourAPI·KOPIS·SDSC 3종 자동 수집 |
| 카카오 로컬 검색 | 장소 검색·지오코딩·지도 표시 |
| 행사 직접 등록 | 소상공인·개인 사진·제목·노출시간 설정 등록 |
| 행사 카테고리·시간 필터 | 오늘·내일·이번 주 / 카테고리별 필터 |
| 체크인·흔적 | 방문 인증·사진·메모 저장, 사용자 페이지 피드 아카이빙 |
| 이음 (친구 연결) | 근거리(100m 이내) 친구 탐색·관계 형성 |
| 지금 패널 | 하단 캡슐 드래그로 현재 흔적·이음 현황 확인 |
| 캡슐 알림 | 파랗게 깜빡이는 캡슐 → 패널 열면 알림 이벤트 자동 이동 |
| 생체 인증 | 지문·얼굴·PIN 인증 (local_auth) |
| 다국어 지원 | 한·영·일·중 4개 언어 |
| 로컬 알림 | 행사 알람·친구 알림 |
| 성인 콘텐츠 분리 | 19세 이상 이벤트 등록 시 성인 인증 필요 |
| 파트너 현황 대시보드 | 이벤트 방문·흔적 통계, 무료이용 현황 |

## 개발 예정 (Firebase 연동 후)

| 기능 | 설명 |
|---|---|
| Firebase 백엔드 연동 | 인증·체크인·알림 서버 저장 |
| 결제 연동 | Naver Pay·Kakao Pay |
| 관심 기능 | 이벤트 저장·시작 알림 |
| Z:GIM ZONE | 지오펜싱 구역 진입 시 무료 등록·기관 협업 |
| 이벤트 환불 정책 | 10% 이내 종료 시 1회 재등록 크레딧 |
